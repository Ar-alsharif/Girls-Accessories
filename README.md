# LunaLace Landing Page

A responsive landing page for a girls accessories store built with HTML, CSS, and vanilla JavaScript.

## Features
- Hero section with call-to-action
- Featured products showcase
- Category browsing cards
- About, testimonials, contact, and footer sections
- Dark/light mode toggle
- Scroll reveal animations and back-to-top button

## Setup
1. Open the project folder in a browser, or serve it locally:
   ```bash
   cd "c:\Users\dell\Downloads\Girls Accessories"
   python -m http.server 3000
   ```
2. Visit http://localhost:3000

## Structure
- `index.html` — main page structure
- `styles.css` — responsive layout and theme styles
- `script.js` — interactivity, theme toggle, and animations
